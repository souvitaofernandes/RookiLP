# Política de Segurança

## Reporte de vulnerabilidades

Se você identificar uma vulnerabilidade de segurança neste repositório, **não abra uma issue pública**.

Entre em contato diretamente pelo e-mail: contato@rooki.com.br
Assunto: `[SECURITY] <descrição breve>`

Responderemos em até 72 horas úteis.

## Escopo

Este repositório contém o código da landing page pública da Rooki.
